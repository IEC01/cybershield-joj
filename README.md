## Démo en direct
> Dashboard SOC cybersécurité avec détection IA des anomalies réseau

## Fonctionnalités
- Surveillance réseau en temps réel via WebSocket
- Moteur IA — Isolation Forest (score anomalie 0 à 1)
- Détection de 4 types d'attaques : DDoS, Brute Force, Port Scan, SQL Injection
- Dashboard sombre avec alertes visuelles et sonores
- Simulateur d'attaques intégré pour les démonstrations
- 4 systèmes JOJ protégés : billetterie, accréditations, résultats, site officiel
